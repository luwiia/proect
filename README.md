Курсовой проект посвящён исследованию промпт-инжиниринга для автоматического извлечения ключевых событий из новостных текстов. В работе сравниваются разные стратегии построения промптов (zero-shot, few-shot и chain-of-thought) и оценивается их влияние на качество выделения субъектов, объектов, действий и времени события.

В качестве моделей используются крупные языковые модели (в том числе Perplexity Sonar и DeepSeek), которым подаются одинаковые новостные тексты в разных вариантах промптов. Полученные ответы автоматически парсятся в структурированный формат с полями subject, object, action, time, после чего проводится анализ качества извлечённых событий и сравнение результатов между моделями и типами промптов.
